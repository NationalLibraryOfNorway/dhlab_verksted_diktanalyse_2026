# Digital diktanalyse: Mønstre i mengden

Velkommen til fagseminar om algoritmisk diktanalyse på Nasjonalbiblioteket 18.februar 2026!

Innholdet i verkstedet er publisert som en Jupyter notebook på Binder, som lar deg kjøre koden interaktivt i nettleseren uten å installere noe selv.

Klikk på knappen eller scan QR-koden for å åpne den interaktive notebooken og prøv koden selv:
